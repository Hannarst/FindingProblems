User Guide

Search Function

The search is a union of all the criteria. The search fields will attempt to autocomplete your query according to the information already avaliable to the database. The following fields are available:

Paradigms

Paradigms are general types of problems. For example: string manipulation.

Algorithms

Search for problems which require a particular algorithm to solve them. For example: binary search, dijkstra's algorithm.

Data Structures

Search for problems with solutions that use a particular data structure. For example: array, stack.

Complexity

Big O notation. Find problems with solutions of a particular complexity. For example: O(nlogn), O(n!)

Languages

Search for problems with solutions in a particular language. For example: Haskell, Python.

Difficulty

Find problems of a particular difficulty.

Visibility

Search for public problems, private (can only be seen by logged in staff) problems or both.

Account Management

Creating an Account and First Time Login

You must have access to a UCT Computer Science staff email address in order to create an account. An email will be sent with an activation code. You will be prompted for this code the next time you attempt to log in. An activated account is required for most of the functionality of the website.
{% if user.is_authenticated and user.is_active %}

Changing and Resetting Password

You may request a password reset from the login page. The system will send a password reset email to the account which you registered yourself on the system with. Once logged in, you may change your password. You will require your current password in order to do this.
